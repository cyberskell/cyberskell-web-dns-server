#Web and DNS Server - cyberskell.com

## Overview 
Configured a Web and DNS server stack on Ubuntu 24 server hosted on virtual box, using the domain (cyberskell.com)


## What was used 
- **Operating System** Ubuntu LTS server 24.04
- **DNS** BIND9 Authoritative name server
- **Web Server:** Apache2 (SSL/TLS) / Virtual hosting
- **Dataase** MySQL 8.0
- **Web Applications** phpMyAdmin and WordPress
- **Security measures** OpenSSL, IP-based access control
- **Domain** cyberskell.com



## Project chapters

2 | BIND9 DNS installation |
